# 💰 Dompetku - Financial Tracker

Aplikasi financial tracker pribadi dengan AI input otomatis.

## Fitur
- ✅ Input transaksi natural language ("kopi 15rb cash")
- ✅ AI auto-klasifikasi kategori (Gemini)
- ✅ Dashboard dengan grafik lengkap
- ✅ Budget bulanan + notif makian lucu
- ✅ Target tabungan + weekly target otomatis
- ✅ Piutang dengan cicilan
- ✅ Dark/light mode otomatis
- ✅ Data tersimpan di Firebase

## Deploy ke Vercel (Gratis)

### 1. Upload ke GitHub
1. Buat repo baru di github.com (nama: `dompetku`)
2. Upload semua file ini ke repo tersebut

### 2. Deploy ke Vercel
1. Buka vercel.com → Log in with GitHub
2. Klik "Add New Project"
3. Pilih repo `dompetku`
4. Klik "Deploy"
5. Tunggu ~2 menit → LIVE! 🎉

### 3. Buka di HP
- Buka URL Vercel di browser HP
- Klik "Add to Home Screen" untuk install seperti aplikasi

## Tech Stack
- React 18
- Firebase Firestore
- Gemini AI API (free tier)
- Recharts
- Deployed on Vercel
